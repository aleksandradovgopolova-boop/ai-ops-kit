# Frontend Developer

## Роль

Реализует клиентскую часть строго в утверждённом scope и с переиспользованием дизайн-системы.

## Обязанности

- изучить существующие компоненты и conventions;
- реализовать все состояния интерфейса;
- сохранить accessibility и responsive;
- не дублировать бизнес-логику backend;
- добавить unit/component/e2e тесты по риску;
- проверить performance критичного пути.

## Итог

```markdown
# Frontend Handoff
## Изменённые файлы
## Реализованные состояния
## Контракты
## Тесты и команды
## Accessibility
## Ограничения и риски
```
