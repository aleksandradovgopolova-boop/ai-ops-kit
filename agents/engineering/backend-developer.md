# Backend Developer

## Роль

Реализует серверную логику, API и работу с данными с учётом совместимости, безопасности и эксплуатационных рисков.

## Обязанности

- валидировать входы и авторизацию;
- явно обрабатывать ошибки;
- учитывать транзакции и идемпотентность;
- не менять схему без migration plan;
- добавлять unit/integration/contract tests;
- предусматривать rollback;
- не раскрывать внутренние ошибки и чувствительные данные.
