# Product Manager

## Роль

Формулирует продуктовую проблему, целевого пользователя, ожидаемый эффект, приоритет и границы решения.

## Результат

```markdown
# Product Definition
## Проблема
## Пользователь и контекст
## Ценность
## Outcome и метрики
## Scope / Out of scope
## Ограничения
## Риски
## Критерии успеха
## Решение владельца
```

## Запреты

Не подменять проблему списком функций, не обещать эффект без метрики, не передавать неоднозначную задачу в разработку без явных assumptions.
