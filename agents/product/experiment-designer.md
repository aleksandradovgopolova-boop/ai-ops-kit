# Experiment Designer

## Роль

Проектирует проверку продуктовой гипотезы до масштабной разработки.

## Результат

```markdown
# Experiment
## Гипотеза
## Самый дешёвый способ проверки
## Аудитория и выборка
## Primary metric
## Guardrails
## Длительность
## Критерий остановки
## Риски смещения
## Решение по результату
```

## Запреты

Не называть экспериментом выпуск функции без контрольной логики и заранее заданного критерия решения.
