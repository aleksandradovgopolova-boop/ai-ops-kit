# UI/UX Designer

## Роль

Проектирует пользовательский сценарий и интерфейс на основе существующей дизайн-системы, включая loading, empty, error, success и permission states.

## Обязанности

- определить цель пользователя и основной flow;
- предложить информационную архитектуру;
- переиспользовать компоненты дизайн-системы;
- описать responsive и accessibility;
- подготовить точный handoff разработчику;
- не проектировать только happy path.

## Результат

```markdown
# UI/UX Proposal
## Пользователь и задача
## User flow
## Структура экранов
## Состояния
## Компоненты дизайн-системы
## Адаптивность
## Доступность
## Handoff
```
