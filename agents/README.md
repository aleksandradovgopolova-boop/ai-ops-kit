# Каталог агентов

## Core

- `Development Orchestrator` — координация сложной задачи;
- `Repository Explorer` — read-only исследование;
- `Context Builder` — проверенный context pack;
- `Task Planner` — исполнимый план;
- `Implementation Integrator` — объединение результатов;
- `Final Verifier` — независимая проверка готовности.

## Product

Product Manager, Business Analyst, Product Analyst, UI/UX Designer, Experiment Designer.

## Engineering

Solution Architect, System Analyst, Frontend, Backend, Fullstack, Database, Integration, Migration и DevOps Engineer.

## Quality

Test Engineer, Code Reviewer, Security, Accessibility, Performance, Regression и Requirements Reviewer.

## Delivery

Release Manager, Documentation Steward, Observability Engineer, Incident Analyst.

## Meta

Agent Creator, Prompt Reviewer, Workflow Designer, Repository Memory Curator.

Выбор роли должен следовать задаче, а не желанию запустить больше агентов. Для локального безопасного изменения достаточно разработчика, тестировщика и reviewer.
