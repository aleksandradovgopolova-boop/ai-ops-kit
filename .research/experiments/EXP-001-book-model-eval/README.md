# EXP-001 — слепое сравнение моделей для Personal Cosmic Book

Проверка рекомендации [DP-101](../../decisions/DP-101.yaml) (`proposed_validation`):
20 одинаковых глав на `deepseek-v4-flash`, `deepseek-v4-pro`, `kimi-k3`;
слепое ревью качества русской прозы + фактическая стоимость.

## Протокол

1. **Генерация** (нужны ключи в env: `DEEPSEEK_API_KEY`, `MOONSHOT_API_KEY`):

   ```bash
   python3 generate.py --dry-run     # план и оценка стоимости (~$1–1.5 на полный прогон, почти всё — kimi-k3)
   python3 generate.py --limit 2     # smoke на 2 главах
   python3 generate.py               # полный прогон, resume поддерживается
   ```

2. **Анонимизация:** `python3 anonymize.py` → `blind/S-*.md` + `mapping.json`
   (судье не показывать до конца оценки).

3. **Слепое ревью.** Судья читает `blind/` и ставит каждому образцу 1–10 по критериям:
   - язык и точность (грамматика, естественность русского);
   - отсутствие штампов и LLM-slop;
   - образность (метафоры работают, а не украшают);
   - следование брифу и правилам стиля (объём, «ты», без буллетов, без эзотерики);
   - целостность главы (композиция, начало-финал);
   - человечность интонации (тепло без лести).
   Оценки — в `scores.yaml` (blind_id → критерии + итог + комментарий).

4. **Раскрытие:** `python3 anonymize.py --reveal`; агрегировать по моделям,
   сравнить с фактической стоимостью из `runs.jsonl`.

5. **Замыкание контура:** результат → `outcome` в DP-101, новые evidence
   (EV-107+: measured_result, качество и стоимость), при смене рекомендации —
   DP-102 со ссылкой superseded.

## Ограничения дизайна

- Главы генерируются независимо (общий контекст книги, но без сквозной памяти
  сюжета) — деградацию на длинной дистанции этот прогон не измеряет; при
  необходимости добавить sequential-режим вторым этапом.
- Судья — LLM (Claude в сессии), не носитель-редактор: вывод о качестве прозы
  считать сильным сигналом, но не финальной истиной; спорные пары образцов
  стоит показать человеку.
- Инвариант writer≠judge соблюдён: модели-кандидаты не участвуют в оценке.
