# EXP-003 — спайк DP-107: build-with-components для ИИ-среды

Проверка принятой рекомендации [DP-107](../../decisions/DP-107.yaml) (`proposed_validation`).
Статус: харнес готов; исполнение частей (1) требует машину с >=16GB RAM для Docker
(Mac Sasha: Docker VM 8GB — не влезает; варианты: RunPod CPU-box ~$0.2-0.4/час, или
поднять Docker до 12GB и гонять компоненты по одному) и LLM-endpoint (стек DP-106 или ключи).

## Часть 1 — спайк (1.5-2 недели инженера; порядок при цейтноте: неделя 1 = ядро, неделя 2 = adopt-замер)

1. `docker compose -f compose.spike.yaml up ragflow presenton` (пиновка образов внутри).
2. Подключить LLM: OpenAI-compatible endpoint (T-lite на vLLM по DP-106; для смоука — любой ключ).
3. Сквозной сценарий: 5-10 обезличенных русских документов → RAGFlow-справка → Presenton-презентация из справки.
4. Пороги (из DP-107): цепочка end-to-end проходит И >=80% цитат справки верифицируемы
   против исходников (adaptировать ../../tools/verify_quotes.py под локальные файлы) И
   презентация не требует ручной пересборки структуры (бинарно).
5. Сетевая изоляция (constraint corporate_data_privacy): во время генерации снять исходящие
   соединения контейнеров — `docker compose exec <svc> sh -c 'netstat -tn'` или tcpdump на
   docker-интерфейсе; любые вызовы наружу, кроме заданного LLM-endpoint, — красный флаг в отчёт.
6. Adopt-замер (вторая неделя, НЕ выбрасывать при цейтноте — на нём висит сравнение путей из
   rationale-2): поднять LibreChat, оценить фактический объём достройки до того же сценария
   (в т.ч. достаточность его логов moderation/token-spend под MVP-грейд журнала) — часы, список дыр.
7. Замер человеко-часов всего спайка — первая точка TCO.

## Часть 2 — интервью (человек; гайд: interview-guide.md)

n=5, стоп-сигнальная семантика: <3 из 5 называют >=2 артефакта триады ценными → СТОП build-пути.
Прохождение порога build НЕ подтверждает (n мал) — подтверждение даст только пилот.

## Часть 3 — юр-проверка WrenAI (юрист)

По дереву исходников https://github.com/Canner/WrenAI: границы AGPL-3.0 компонентов против
Apache-2.0 из README; вердикт: можно ли линковаться/деплоить как отдельный сервис без
заражения. До вердикта WrenAI в компонентную схему не входит.
