# Repository Memory

Хранит только устойчивые знания, полезные будущим задачам.

- `decisions/` — ADR и важные product/technical decisions;
- `patterns/` — проверенные reusable patterns;
- `incidents/` — выводы из инцидентов;
- `known-issues/` — подтверждённые ограничения;
- `lessons-learned/` — повторяемые уроки.

Каждая запись содержит источник, owner, дату проверки и условие устаревания.
