# Tool Usage

- Использовать минимально необходимые права.
- Production и внешние write-действия требуют явного разрешения.
- Команды должны быть воспроизводимыми и записываться в evidence.
- Ошибки инструментов не скрываются.
- Не использовать генерацию данных или mocks так, чтобы они маскировали реальный сценарий.
