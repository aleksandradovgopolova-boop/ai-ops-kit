# Scope Control

- Изменяются только необходимые файлы.
- Новые зависимости требуют обоснования и approval по уровню риска.
- Массовое форматирование отделяется от функционального изменения.
- Соседние дефекты записываются отдельно.
- Расширение scope требует обновления TaskPlan.
- Protected paths нельзя менять без владельца.
- Агент обязан остановиться, если обнаруженный объём существенно превышает план.
