# Source of Truth

| Тип информации | Канонический источник |
|---|---|
| Функциональное поведение | Код + acceptance criteria |
| API | Контракт / OpenAPI |
| Архитектурное решение | ADR |
| Бизнес-правила | Product context |
| Схема данных | Миграции / schema |
| Статус задачи | TaskState |
| Эксплуатация | Runbook |

При конфликте агент фиксирует расхождение и не выбирает источник молча. Приоритет задаётся локальными правилами владельца системы.
