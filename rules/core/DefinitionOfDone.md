# Definition of Done

Задача завершена, когда:

- выполнены и доказаны acceptance criteria;
- scope не нарушен;
- build, lint, types и tests пройдены по применимости;
- review завершены;
- миграция, rollback и monitoring определены по риску;
- источники истины обновлены;
- непроверенные области и исключения явно зафиксированы;
- Final Verifier выдал passed или approved passed_with_exceptions.
