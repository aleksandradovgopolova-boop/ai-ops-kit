# Evidence Policy

Агент не имеет права утверждать, что:

- тесты прошли, если он их не запускал;
- файл или логика отсутствуют, если не выполнил поиск;
- API поддерживает поле без проверки контракта;
- ошибка исправлена только потому, что изменён код;
- задача готова без сопоставления критериев и evidence.

Evidence должно быть воспроизводимым: команда, результат, ссылка на diff, тест, скриншот или подтверждённый ручной сценарий.
