# Quality Gates

## Analysis

- требования проверяемы;
- scope определён;
- affected components известны.

## Implementation

- build/lint/types/tests пройдены;
- нет неутверждённых зависимостей;
- контракты и миграции согласованы.

## Review

- code review завершён;
- risk-based reviews завершены;
- blockers закрыты.

## Delivery

- rollback и monitoring определены;
- документация обновлена;
- evidence приложено.
