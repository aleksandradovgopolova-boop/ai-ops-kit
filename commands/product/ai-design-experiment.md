# Команда: ai-design-experiment

## Назначение

Проверить гипотезу наиболее дешёвым валидным способом.

## Порядок выполнения

1. Сформулировать гипотезу.
2. Выбрать метод проверки.
3. Определить выборку и метрики.
4. Зафиксировать критерий решения до запуска.
5. Проверить риски смещения.

## Результат

Experiment document.
