# Команда: ai-create-feature

## Назначение

Сформировать проверяемое описание функции.

## Порядок выполнения

1. Связать feature с outcome.
2. Описать user flow и правила.
3. Зафиксировать states, permissions и edge cases.
4. Добавить метрики и acceptance criteria.
5. Провести Requirements Review.

## Результат

Feature document.

## Ограничения

- Соблюдать scope, evidence policy и human approvals.
