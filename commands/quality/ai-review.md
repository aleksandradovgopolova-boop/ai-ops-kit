# Команда: ai-review

## Назначение

Запустить набор независимых review по риску изменения.

## Порядок выполнения

1. Всегда запустить Code Reviewer.
2. Подключить Security Reviewer для данных, доступов и интеграций.
3. Подключить Accessibility Reviewer для UI.
4. Подключить Performance Reviewer для критичных путей.
5. Объединить findings без потери приоритета.

## Результат

Сводный review verdict.

## Ограничения

- Соблюдать scope, evidence policy и human approvals.
