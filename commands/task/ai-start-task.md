# Команда: ai-start-task

## Назначение

Единая точка входа для новой задачи.

## Порядок выполнения

1. Создать или проверить `TaskBrief`.
2. Запустить `Repository Explorer` и `Context Builder`.
3. Классифицировать задачу по типу, сложности и риску.
4. Выбрать workflow.
5. Запустить `Requirements Reviewer`, если требования неоднозначны.
6. Создать `TaskPlan`.
7. Запросить только обязательные approvals.
8. Инициализировать `TaskState`.

## Результат

TaskContext, TaskPlan, TaskState и выбранный workflow.

## Ограничения

Не начинать реализацию до подтверждения блокирующих решений.
