# Команда: ai-clarify-task

## Назначение

Снизить неоднозначность задачи без допроса пользователя о том, что уже известно репозиторию.

## Порядок выполнения

1. Исследовать код, документацию и context.
2. Закрыть вопросы, на которые уже есть ответ.
3. Выделить блокирующие неизвестные.
4. Предложить безопасные рабочие assumptions.
5. Обновить TaskBrief.

## Результат

Список подтверждённых решений, assumptions и действительно открытых вопросов.

## Ограничения

- Соблюдать scope, evidence policy и human approvals.
