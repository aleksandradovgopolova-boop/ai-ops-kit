# Команда: ai-verify

## Назначение

Проверить доказательства выполнения критериев.

## Порядок выполнения

1. Сопоставить каждый критерий с проверкой.
2. Запустить доступные команды.
3. Зафиксировать результаты и непроверенные области.
4. Запустить Final Verifier.
5. **Записать срез эффекта**: `tools/run_report.py features/<id> --record` (автонакопление истории).

## Результат

VerificationEvidence и финальный вердикт.

## Ограничения

Не считать отчёт другого агента доказательством без воспроизводимой проверки.
