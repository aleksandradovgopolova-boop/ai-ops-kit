# Команда: ai-implement

## Назначение

Реализовать утверждённый план.

## Порядок выполнения

1. Прочитать TaskState, план и локальные правила.
2. Проверить разрешённые файлы.
3. Выполнить назначенный work package.
4. Добавить тесты.
5. Запустить доступные проверки.
6. Подготовить TaskHandoff и evidence.
7. **Записать срез эффекта**: `tools/run_report.py features/<id> --record` (автонакопление истории).

## Результат

Код, тесты, handoff и результаты проверок.

## Ограничения

Не менять файлы вне write-scope и не обходить failed checks.
