# Команда: ai-discover

## Назначение

Read-only исследование реализации или области репозитория.

## Порядок выполнения

1. Сформулировать вопрос.
2. Прочитать локальные правила и карту зоны.
3. Найти точки входа, зависимости, тесты и источники истины.
4. Вернуть компактную карту и риски.

## Результат

Repository Map без изменений файлов.

## Ограничения

Запрещено редактирование и выдача предположений за факты.
