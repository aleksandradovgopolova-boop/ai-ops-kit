# Команда: ai-plan-task

## Назначение

Подготовить исполнимый план без реализации.

## Порядок выполнения

1. Проверить context и acceptance criteria.
2. Определить минимальное решение.
3. Создать work packages.
4. Назначить write-scope, проверки и зависимости.
5. Определить approvals, rollback и gates.

## Результат

Заполненный TaskPlan.

## Ограничения

Не писать код и не расширять scope.
