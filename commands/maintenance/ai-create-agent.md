# Команда: ai-create-agent

## Назначение

Создать новую роль с оценимым результатом.

## Порядок выполнения

1. Доказать, что существующая роль недостаточна.
2. Описать trigger, scope, inputs, outputs и prohibitions.
3. Создать файл по шаблону.
4. Добавить минимум три evaluation cases.
5. Обновить индекс.

## Результат

Новый агент и evaluations.

## Ограничения

- Соблюдать scope, evidence policy и human approvals.
