# Команда: ai-fix-bug

## Назначение

Диагностировать и исправить дефект с минимальным scope.

## Порядок выполнения

1. Воспроизвести или подтвердить симптом.
2. Найти root cause.
3. Определить regression scope.
4. Добавить падающий тест, если возможно.
5. Исправить причину.
6. Запустить targeted и regression checks.

## Результат

Bug report, fix, tests, evidence.

## Ограничения

Не маскировать симптом fallback-логикой без устранения причины.
