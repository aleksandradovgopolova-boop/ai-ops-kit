# Eval-кейсы агентов

Файл на агента: `<agent-id>.md` (id = имя файла агента без расширения, например
`task-planner.md` для `agents/core/task-planner.md`). Минимум 3 кейса по шаблону
`../AgentEvaluationCase.md`. Наличие файла для изменённых агентов проверяет CI
(`validation/validate_agent_evals.py`).
