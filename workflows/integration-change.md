# Workflow: интеграционное изменение

```text
Context Builder
→ System Analyst
→ Integration Engineer
→ Solution Architect
→ Security Reviewer
→ Human approval
→ Backend/Integration implementation
→ Contract tests
→ Observability Engineer
→ Release Manager
→ Final Verifier
```

Обязательны versioning, retries/timeouts, idempotency, degradation strategy, audit и rollback.
